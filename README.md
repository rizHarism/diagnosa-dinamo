# Sistem Pakar Diagnosa Dinamo Pengisian

Sistem pakar sederhana untuk mendiagnosa kerusakan pada dinamo pengisian motor/mobil menggunakan metode Forward Chaining.

## Fitur

- Input gejala interaktif
- Sistem diagnosa berbasis rule/aturan
- Implementasi Forward Chaining

## Cara Menggunakan

1. Buka `index.html` di browser
2. Pilih gejala yang diketahui
3. Klik tombol "Diagnosa"
4. Lihat hasil diagnosa

## Deployment

Proyek ini bisa di-deploy secara static hosting seperti Vercel, Netlify, atau GitHub Pages.

## Teknologi

- HTML5
- CSS3
- JavaScript ES6
